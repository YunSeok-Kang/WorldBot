# 개발 상태
현재는 WorldBot내에 SlackTalker에서 해야할 기능을 구현해놨다. 직접 메시지를 받아버리는 상태. 하지만 슬랙과 통신하는 부분은 모두 뺄 생각.

Commands에 있는 세부 명령은 현재 동작하지 않는 상태. /WAI 만 가능.
